Class UFTextFileLog

Namespace
UltraForce.Library.NetStandard.Development
Assembly
UltraForce.Library.NetStandard.dll

UFTextFileLog implements a log that is written to a text file. New messages get added to the end of the log file. The messages are prefixed with a time stamp.

The class will create a a new log file when the day changes. The log filename includes a date.

If UFDEBUG is defined, the log messages are also outputted to the general debugger log via WriteLine(string).
