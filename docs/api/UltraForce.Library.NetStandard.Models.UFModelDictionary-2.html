Remarks

UFModelDictionary<TKey, TValue> will fire a DataChanged event whenever the contents of the dictionary changes.

If LockChildren option is set and TValue implements the IUFLockable interface, Lock() will also lock all items within the list.

If TrackChildChange option is set and TValue implements IUFNotifyDataChanged, UFModelDictionary<TKey, TValue> will install change listeners with every stored value and will fire DataChanged, ChildChanged and PropertyChanged events if a value changes.

UFModelDictionary<TKey, TValue> will make sure to install only one listener, even if the same value instance is added for multiple keys.

The default implementation sets all the properties that have a DefaultValueAttribute attached to the value specified by that attribute.

As last step the method will call UpdateTrackedProperties()
