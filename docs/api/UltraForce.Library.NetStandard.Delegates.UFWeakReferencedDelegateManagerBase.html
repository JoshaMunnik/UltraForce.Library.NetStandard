UFWeakReferencedDelegateManagerBase is a base class for classes that need to manage a group of weak referenced delegates using a UFWeakReferencedDelegateBase subclass.

It is an abstract class and should never be instantiated directly.

Add(UFWeakReferencedDelegateBase)

Adds a delegate to the managed list. If there is already a delegate with the same method and target stored, nothing happens.

The method tries first to replace an entry that has a target that is no longer available before adding it to the end of the list.
