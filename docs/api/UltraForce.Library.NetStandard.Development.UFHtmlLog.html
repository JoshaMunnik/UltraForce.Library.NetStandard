Class UFHtmlLog

Namespace
UltraForce.Library.NetStandard.Development
Assembly
UltraForce.Library.NetStandard.dll

UFHtmlLog implements a log using HTML formatting for every entry. New messages get added to log and are output to the general debugger log via WriteLine(string).

This class only has functionality if UFDEBUG has been defined else the methods will do nothing.
