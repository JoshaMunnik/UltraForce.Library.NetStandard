Class UFExternalAction

Namespace
UltraForce.Library.NetStandard.Controllers.Actions

UltraForce.Library.NetStandard.dll

UFExternalAction can be used to implement actions that start something and have to wait for it to finish (for example via a callback).

RunAsync(CancellationToken) will call StartAsync(CancellationToken) and then waits until a call is made to Done(bool).
